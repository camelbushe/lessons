# Хэш таблицы
Структура данных, которая хранит элементы в виде пар ключ-значение.
## Что такое хэш функция
**Хэш функция** - это функция, которая принимает строку и возвращает число, которое будет соответствовать этой строке.
### Требования
- Для одной и той же строки всегда возвращает одно и тоже число.
- Одно и тоже число не может соответствовать ещё каким либо строкам.
## Суть
Хэш таблиц не существует в компьютерах. Это что-то абстрактное, но вот как она реализуется.
#### При добавлении
1. Принимаем строку
2. Пропускаем через хэш функцию
3. Сохраняем в массиве значение для соответствующей строки под индексом, который вернула функция.
#### При возврате
1. Принимаем строку
2. Пропускаем через хэш функцию
3. Обращаемся к массиву по индексу который вернула функция и получаем значение для переданной строки.
## Хэш таблицы в языках программирования
- В JavsScript это объекты
- В Python это словари
